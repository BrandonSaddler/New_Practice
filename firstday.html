Computer

A computer is an electronic device that processes and stores data. It consists of various hardware components and software programs that work together to perform tasks.

Hard drive

Hard drive is a storage medium used to store data.

Hard Disk Drive (HDD)

A Hard Disk Drive (HDD) is a traditional storage device used in computers. It uses spinning magnetic disks to store data. HDDs are known for their large storage capacity but are slower compared to Solid State Drives (SSDs).

Solid State Drive (SSD)

A Solid State Drive (SSD) is a newer type of storage device that uses flash memory to store data. SSDs are faster, more reliable, and energy-efficient compared to HDDs. They are commonly used in modern computers for improved performance.

Both HDDs and SSDs are essential components of a computer's storage system, with each having its advantages and disadvantages.

RAM

RAM stands for random access memory. It is called “random access” because the technology allows the computer to temporarily store and access data immediately.

DDR2

Double Data Rate 2 Synchronous Dynamic Random-Access Memory is a double data rate synchronous dynamic random-access memory interface.

DDR3

Double Data Rate 3 Synchronous Dynamic Random-Access Memory is a type of synchronous dynamic random-access memory with a high bandwidth interface, and has been in use since 2007.

DDR4

Double data rate fourth generation (DDR4) is a memory standard designed as a better, faster, more reliable replacement for DDR3.
